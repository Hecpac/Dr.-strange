# HEARTBEAT.md — Echo 🔊

# Echo is passive by default — does NOT auto-publish, auto-comment, or auto-engage.
# Heartbeat is observability + drafting suggestions only.

## Checks (rotate, don't do all every time)

### Priority 1 — Every heartbeat
- [ ] New comments on Hector's last 3 IG / LinkedIn / X posts that need a reply draft?
- [ ] Any draft caption Hector left half-written in `artifacts/content/`?

### Priority 2 — Every 2-3 heartbeats
- [ ] Has the weekly Tuesday post been drafted by Sunday PM? (Cadence compromise)
- [ ] Any competitor in the watchlist published something worth a counter-take?

### Priority 3 — Once per day
- [ ] Engagement audit: which of the last 7 days' posts landed (>median reach), which flopped?
- [ ] Hashtag fatigue check: are we reusing the same 5 hashtags 7 days in a row?

## Rules

- Quiet hours: 23:00–08:00 CT (HEARTBEAT_OK unless engagement signal is critical — e.g. a viral comment thread or impersonation)
- If nothing needs attention → HEARTBEAT_OK
- NEVER auto-publish from heartbeat. Drafts go to a queue, Hector approves.
- NEVER engage with hostile / political / off-topic comments without explicit ask from Hector.
- Anti-shadowban floor: if account is <30 days old, max 1 post draft per day, no DM blasts, no follow sprees.
