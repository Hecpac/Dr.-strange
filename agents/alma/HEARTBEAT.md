# HEARTBEAT.md — Alma 🔮

# Cron already handles: morning-brief (8 AM)
# Heartbeat covers: between-brief awareness

## Checks (rotate, don't do all every time)

### Priority 1 — Every heartbeat
- [ ] Unread Telegram messages needing follow-up?
- [ ] Calendar events in the next 2 hours?

### Priority 2 — Every 2-3 heartbeats
- [ ] Unread emails flagged urgent or from known contacts?
- [ ] Any pending items Hector asked you to remind him about?

### Priority 3 — Once per day
- [ ] Weather if Hector might be heading out
- [ ] Review today's memory file — anything worth promoting to MEMORY.md?

## Rules
- Quiet hours: 23:00–08:00 CT (HEARTBEAT_OK unless urgent)
- If nothing needs attention → HEARTBEAT_OK
- Telegram messages: keep under 3 sentences
- Spanish or English based on Hector's last language
